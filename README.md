# 🛡️ Splendor: Elemental Masters

**Splendor: Elemental Masters**는 고전 보드게임 '스플렌더'를 현대적인 감각과 판타지 테마로 재해석한 웹 기반 로컬 멀티플레이어 보드게임입니다. 플레이어는 원소의 마스터가 되어 정수를 수집하고, 고대의 마법서를 구매하며, 위대한 귀족 보호자들의 선택을 받아 최고의 마스터가 되어야 합니다.

---

## 🎮 게임 설명

각 플레이어는 자신의 턴에 세 가지 행동 중 하나를 수행할 수 있습니다:
1.  **원소 정수(Token) 수집**: 서로 다른 색상의 정수를 3개 가져오거나, 4개 이상 쌓인 같은 색상의 정수를 2개 가져옵니다.
2.  **마법서(Card) 예약**: 보드의 마법서 하나를 예약하고 황금 정수(조커) 1개를 획득합니다. (최대 3장)
3.  **마법서(Card) 구매**: 보유한 정수를 지불하여 보드 혹은 예약된 마법서를 영구적으로 소유합니다.

**승리 조건**: 마법서와 귀족으로부터 점수를 획득하여 가장 먼저 **15점**에 도달하는 플레이어가 승리합니다. (15점 도달 시 해당 라운드의 마지막 플레이어까지 턴을 진행한 후 게임이 종료됩니다.)

---

## ✨ 주요 기능

-   **완전한 게임 루프**: 토큰 수집, 카드 예약, 구매, 귀족 방문 등 모든 핵심 규칙 구현.
-   **실시간 점수 및 보너스 계산**: 구매한 카드가 제공하는 영구 보너스가 다음 카드 구매 시 자동으로 적용됩니다.
-   **지능형 UI**: 구매 가능한 카드를 시각적으로 강조(`Ready` 배지)하고, 획득 가능한 귀족을 자동으로 체크합니다.
-   **승리 판정 로직**: 15점 도달 시 라운드 종료 및 승자 선언 로직(동점 시 카드 적은 사람 승리) 구현.
-   **반응형 디자인**: 다양한 화면 크기에서도 쾌적하게 플레이할 수 있는 UI.
-   **로컬 저장 기능**: `Zustand Persistence`를 통해 새로고침을 해도 게임 상태가 유지됩니다.

## 🛠️ 기술 스택

-   **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
-   **Library**: [React 19](https://react.dev/)
-   **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
-   **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
-   **Utilities**: `clsx`, `tailwind-merge` (고급 클래스 관리)

---

## 🏗️ 아키텍처 및 리팩토링

본 프로젝트는 유지보수와 확장성을 위해 다음과 같은 설계 원칙을 준수합니다:
-   **Atomic Design**: 컴포넌트를 최소 단위로 분리하여 재사용성과 가독성을 높였습니다.
-   **Centralized State**: 게임의 모든 핵심 로직은 `useGameStore`에서 중앙 집중식으로 관리됩니다.
-   **Clean UI Class Management**: `cn()` 유틸리티를 사용하여 테일윈드의 긴 클래스 문자열을 체계적으로 관리합니다.
